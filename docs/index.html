What it does

Everything a first conversation needs

MentalBridge was designed around one insight — the hardest step is saying the words out loud for the very first time.

💬

Warm AI Chat

Gemma 4 responds warmly in your language. Short sentences. One gentle question. No clinical jargon. Like a trusted older sibling.

📵

100% Offline

Everything runs on your device via Ollama. Turn off WiFi — it still works. Your words never leave your laptop. Ever.

🗣️

Multilingual

Tamil, Hindi, Telugu, Kannada, English. Type in your language and get a response in the exact same language automatically.

🚨

Crisis Detection

Instantly detects distress signals and surfaces Indian helplines — iCall, Vandrevala Foundation — not American numbers.

🫧

Breathing Exercise

Guided 4-7-8 breathing with animated real-time visual. Clinically proven to reduce acute anxiety in minutes.

🌊

Calm Space

Ripple Pond wave physics, Pop Your Thoughts (ACT therapy), and 5-4-3-2-1 Grounding — all evidence-based techniques.

📊

Mood Tracker

Log your mood daily with a note. Track patterns over time to understand your emotional landscape better.

🔒

Privacy First

No account. No login. No analytics. No data collection. The privacy is not a feature — it is the entire point.

Setup guide

Get running in 4 steps

Choose your operating system:

🍎 macOS
🪟 Windows
1

Install Ollama

Download and install Ollama — the engine that runs Gemma 4 locally on your Mac.

Download Ollama for Mac →
2

Download Gemma 4 E2B

Open Terminal (search "Terminal" in Spotlight) and run this command. It downloads Gemma 4 — about 7.2GB, takes a few minutes.

ollama pull gemma4:e2b
3

Download MentalBridge

Download the ZIP file, unzip it. Then right-click the unzipped folder and select "New Terminal at Folder". Run:

bash START.sh
⬇️ Download ZIP
4

Open in browser

MentalBridge opens automatically in your browser. If it doesn't, open Chrome or Safari and go to:

http://localhost:8080
✅ You're all set. MentalBridge runs completely offline from this point. No internet needed.
1

Install Ollama

Download and install Ollama for Windows. Run the installer and follow the setup wizard.

Download Ollama for Windows →
2

Download Gemma 4 E2B

Open Command Prompt (search "cmd" in Start menu) and run:

ollama pull gemma4:e2b
⚠️ This downloads 7.2GB. Make sure you have enough disk space and a stable internet connection.
3

Download MentalBridge

Download the ZIP file and unzip it. Then open Command Prompt, navigate to the folder and run:

cd path\to\mentalbridge_download
python -m http.server 8080

In a second Command Prompt window run:

set OLLAMA_ORIGINS=* && ollama serve
⬇️ Download ZIP
💡 Python required. If you don't have Python, download it from python.org and make sure to check "Add to PATH" during installation.
4

Open in browser

Open Chrome or Edge and go to:

http://localhost:8080
✅ You're all set. MentalBridge runs completely offline from this point.
